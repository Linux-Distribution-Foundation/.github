# Linux Distribution Foundation Code of Conduct

The Linux Distribution Foundation (LDF) is committed to maintaining a respectful, welcoming, and professional community.

This Code of Conduct applies to all LDF spaces, including repositories, issues, pull requests, and discussions.

---

## 1. Expected Behavior

Participants must:

- Treat others with respect and professionalism  
- Communicate constructively and avoid personal attacks  
- Assume good intent and seek clarification rather than escalating  
- Follow project maintainers’ decisions  
- Use inclusive and clear language  

---

## 2. Unacceptable Behavior

The following behaviors are prohibited:

- Harassment, discrimination, or derogatory comments  
- Personal insults or inflammatory behavior  
- Persistent disruption of discussions  
- Sharing others’ private information  
- Submitting intentionally harmful or deceptive contributions  

---

## 3. Enforcement

Project maintainers may:

- Request behavioral corrections  
- Remove inappropriate content  
- Temporarily or permanently ban participants  
- Close issues or PRs that violate this Code  

Maintainers’ decisions are final.

---

## 4. Reporting

The Foundation does not maintain a centralized contact address at this time.  
To report a violation:

- Open an Issue or Discussion in the relevant repository, **if appropriate**, or  
- Contact a project maintainer directly via GitHub  

Private communication channels may be added in the future.

---

## 5. Scope

This Code of Conduct applies to:

- All LDF repositories  
- All interactions within GitHub under the Linux Distribution Foundation organization  
- Any community spaces officially managed by the LDF  

---

We thank all contributors for helping maintain a positive and productive environment.
